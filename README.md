# OrderFlow Management POC

This project is a scalable proof-of-concept for order flow management tailored for small businesses (25–50 employees), with future scalability in mind.

## Tech Stack
- Backend: Python FastAPI
- Frontend: React.js (Material UI)
- Database: PostgreSQL
- Authentication: JWT + Mobile OTP

## Features
- Multi-role user management (Admin, Manager, Employee, Customer)
- Order creation, assignment, tracking, and status updates
- Secure authentication (login + mobile OTP)
- Modern, intuitive UI
- Scalable architecture for future expansion

## Getting Started
1. Clone the repository
2. Install backend and frontend dependencies
3. Configure environment variables
4. Run backend and frontend servers

## Folder Structure
- `/backend` — FastAPI backend
- `/frontend` — React.js frontend
